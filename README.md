## â–¶ï¸ How to Run

1. Clone this repository:
```bash
git clone https://github.com/taehwanlee83/SIS-Advisor.git
cd sim-gen-sis
```
2. Adjust the number of repeatition (default: 100) before you run.
Starting with less than 10 is recommended since it will take some time for each repeatition depending on your running environment.

3. Run the main script:
```
python code/Automated_SSB+IMB.py
```

4. The script will generate `exp_result.csv` in the project root

5. Make you own analysis using generated `exp_result.csv` file.
